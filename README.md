# Blast Furnace Digital Twin & Forecast Dashboard

## Project Overview

This project develops a Digital Twin for Blast Furnace operation using Streamlit, Machine Learning, and Power BI. It enables real-time process simulation, KPI monitoring, forecasting, and operator decision support through an interactive dashboard.

---

## Features

- Digital Twin Simulation
- Blast Furnace KPI Monitoring
- Hot Metal Production Forecasting
- Machine Learning Model Comparison
- Operator Remarks & Recommendations
- Interactive Power BI Dashboard
- Historical Trend Analysis
